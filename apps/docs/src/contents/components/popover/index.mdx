## Close On Escape

`closeOnEscape` prop을 `false`로 설정하면 escape 키로 팝오버가 닫히지 않습니다.

<CodeBlock preview={<CloseOnEscape />} code={CloseOnEscapeRaw} />

## Close On Outside Click

`closeOnOutsideClick` prop을 `false`로 설정하면 외부 클릭으로 팝오버가 닫히지 않습니다.

<CodeBlock preview={<CloseOnOutsideClick />} code={CloseOnOutsideClickRaw} />

## Close On Both

`closeOnEscape` prop과 `closeOnOutsideClick` prop을 모두 `false`로 설정하면 escape 키와 외부 클릭으로 팝오버가 닫히지 않습니다.

<CodeBlock preview={<CloseOnBoth />} code={CloseOnBothRaw} />

## Disable Animation

`disableAnimation` prop을 `true`로 설정하면 팝오버의 애니메이션이 비활성화됩니다.

<CodeBlock preview={<DisableAnimation />} code={DisableAnimationRaw} />
