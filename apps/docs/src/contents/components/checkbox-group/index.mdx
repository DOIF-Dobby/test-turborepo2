# Checkbox Group

## Overview

> `CheckboxGroup`은 여러 개의 `Checkbox`를 그룹으로 묶어 관리할 때 사용합니다.
> `Checkbox` 컴포넌트와 달리, `CheckboxGroup`은 **배열 형태의 값**을 관리하며, 내부의 `Checkbox`들은 그룹의 상태를 공유합니다.
